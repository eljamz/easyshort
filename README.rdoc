== - easyshort -

easyshort - The easy way to short your URL's.

It has been developed with:

* Ruby 1.9.3

* Rails 4.1.5

* MongoDB - mongoid

Feel free to copy it and modify. Just clone the reposiroty and:

* bundle insall

* rake db:mongoid:create_indexes

* rake db:seed

And it will fill you database with 2 example users: (easyshort.com does't exists)

* admin@easyshort.com // please2014

* user2@easyshort.com // please2014
